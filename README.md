height:100%不起作用
问题：继承了之前的全局属性
CSS属性是有继承性的，而百分比都又是相对的，那么height：100%就是相当于容器父级而言的。所以并不是这个高度100%没有效果，只是没有达到我们预期的效果，我们理解上的错误。

理解了高度100%的意义，剩下的就简单了，给body一个100%高度即可。
